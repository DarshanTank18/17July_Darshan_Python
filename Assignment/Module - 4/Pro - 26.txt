Explain Inheritance in Python with an example? 

--> Inheritance relationship defines the classes that inherit from other classes as derived, subclass, or sub-type classes.

--> There are three typea of inheritance:
1) Single inheritance
2) Multiple Inheritance
3) Multilevel Inheritance

--> Example : 
Class BaseClass:
    {Body}
Class DerivedClass(BaseClass):
    {Body}

What is init? 

--> The __init__ method is often referred to as a constructor. 
--> It is a special method that gets called automatically when an object of a class is created.

Or What Is A Constructor In Python?

--> a constructor is a special method called when an object is created.
